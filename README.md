# Weather App

A simple weather application built with HTML, CSS, and JavaScript.

## Features

- Real-time weather data for any city
- emperature unit toggle (Celsius/Fahrenheit)
- Search history (last 5 cities)

## How to Use

1. Enter a city name in the search box
2. Click "Search" or press Enter
3. View current weather, temperature, humidity, and wind speed
4. Toggle between °C and °F
5. Click on recent searches to quickly view weather again

## Setup Instructions

1. Clone this repository
2. Get a free API key from [OpenWeatherMap](https://openweathermap.org/api)
3. Create a `config.js` file in the root directory:
```javascript
   const API_KEY = 'your_api_key_here';
```
4. Open `index.html` in your browser

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+)
- OpenWeatherMap API
- localStorage for search history
